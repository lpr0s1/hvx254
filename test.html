Terminal iOS

Mise à jour de base

apk upgrade

Outils réseau et Python

apk add nmap curl wget openssh python3 py3-pip

sqlmap via pip

pip3 install --upgrade pip

Android • Termux

Terminal puissant sur Android, parfait pour un kit de tests rapide.

Terminal Android

Mise à jour Termux

pkg update && pkg upgrade

Outils réseau et bruteforce

pkg install nmap hydra sqlmap git python

Cloner un toolkit

git clone https://github.com/example/pentest-toolkit.git
cd pentest-toolkit
python tool.py

Windows • PowerShell & CMD

Utilise winget ou Chocolatey pour installer rapidement tes outils.

Desktop

winget en PowerShell

winget install -e --id Nmap.Nmap
winget install -e --id WiresharkFoundation.Wireshark

Installer Chocolatey

Set-ExecutionPolicy Bypass -Scope Process -Force;
[System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072;
iex ((New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))

Installer les outils via choco

choco install nmap wireshark python -y
