Pentest • Launcher

Pentest tools • noir & blanc

Page minimaliste pour installer rapidement quelques outils de pentest sur iOS (ish), Android (Termux) et Windows. Utilisation strictement légale et éthique uniquement.

Paramètre d’URL supporté : ?tool=ios | android | windows

iOS • ish (Alpine)

Terminal iOS

Sur ish, tu es dans un environnement type Alpine Linux. Tu peux utiliser apk pour installer des outils.

Mettre à jour les paquets

apk update
apk upgrade

Installer quelques outils de base

# nmap (scan réseau)
apk add nmap

# curl, wget
apk add curl wget

# openssh client
apk add openssh

# python3 + pip
apk add python3 py3-pip

Exemple : installer sqlmap via pip

pip3 install --upgrade pip
pip3 install sqlmap

Android • Termux

Terminal Android

Sur Termux, tu utilises pkg (ou apt) pour installer des outils de pentest.

Mettre à jour Termux

pkg update && pkg upgrade

Installer quelques outils

# nmap
pkg install nmap

# hydra (bruteforce, à utiliser légalement)
pkg install hydra

# sqlmap
pkg install sqlmap

# git, python
pkg install git python

Cloner un repo d’outils (exemple générique)

git clone https://github.com/example/pentest-toolkit.git
cd pentest-toolkit
python tool.py

Windows • PowerShell & CMD

Desktop

Sur Windows, tu peux utiliser PowerShell avec winget ou Chocolatey, ou encore CMD classique.

PowerShell • via winget

# Ouvrir PowerShell en admin puis :
winget install -e --id Nmap.Nmap
winget install -e --id WiresharkFoundation.Wireshark

PowerShell • installer Chocolatey

Set-ExecutionPolicy Bypass -Scope Process -Force;
[System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072;
iex ((New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))

CMD / PowerShell • installer des outils via Chocolatey

choco install nmap -y
choco install wireshark -y
choco install python -y
